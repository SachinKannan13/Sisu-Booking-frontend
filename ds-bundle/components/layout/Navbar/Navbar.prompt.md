Navbar from booksphere-client. Use via `window.Booksphere.Navbar` (bundle loaded from the root `_ds_bundle.js`). Wrap the tree in `<BooksphereProvider>` (full provider chain in README.md — components read theme/i18n from that context).

## Examples

### Default

```jsx
() => (
  <div style={{ background: '#0d0d0d', minHeight: '56px' }}>
    <Navbar />
  </div>
)
```
